# Roadmap

CurveStudio is an MVP-stage macOS project for offset print calibration curve work.

## Phase 1: Project foundation

- Improve README and project description
- Add open-source license
- Add example measurement datasets
- Add workflow documentation
- Add screenshot guidelines
- Prepare the repository for public visibility

## Phase 2: Core calculation workflow

- Add CMYK measurement data model
- Add tone-value table editor
- Add TVI calculation
- Add target comparison
- Add compensation value calculation
- Add interpolation for corrected input values
- Add basic curve visualization

## Phase 3: Import and export

- Add CSV import
- Add CSV export
- Add example templates
- Export corrected tone tables
- Add validation for missing or inconsistent measurement points

## Phase 4: Production workflow support

- Save calibration projects
- Add press and paper metadata
- Add ink and plate metadata
- Add curve history
- Add verification comparison
- Add notes for pressroom conditions

## Phase 5: RIP workflow preparation

- Add RIP-oriented export presets
- Add channel-specific output tables
- Add smoothing controls
- Add maximum correction limits
- Add manual point locking

## Phase 6: Quality and maintenance

- Add automated tests for curve calculation logic
- Add test datasets
- Add documentation examples
- Add GitHub issue templates
- Add release notes

## Long-term goal

The long-term goal is to make CurveStudio a transparent, practical, and maintainable open-source tool for printing companies and color management technicians who need to calculate, review, and document print compensation curves.

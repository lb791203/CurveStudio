# Contributing to CurveStudio

Contributions are welcome, especially in the following areas:

- macOS and SwiftUI implementation
- Print calibration workflow design
- TVI / CTV / G7 curve calculation logic
- CSV import and export
- Curve visualization
- Documentation and example datasets
- Test cases for compensation calculations

## Development notes

Please keep the project focused on practical offset printing calibration workflows. New features should make measurement input, curve analysis, compensation calculation, or RIP curve preparation clearer and more reliable.

## Data and privacy

Do not commit private customer data, production files, or measurement reports that identify a real customer without permission. Use anonymized or example datasets whenever possible.

## Documentation

When adding a feature, update the README or relevant documentation in `docs/` if the workflow changes.

# Screenshots

Place CurveStudio application screenshots in this directory.

Recommended files:

- `main-window.png` — main application window
- `curve-analysis.png` — CMYK curve analysis view
- `data-table.png` — measurement data table
- `export-panel.png` — export or output panel

## Screenshot guidelines

- Use real screenshots from the running macOS app.
- Avoid showing private customer names or private production data.
- Use example datasets from the `examples/` directory when possible.
- Keep screenshots clear and readable on the GitHub project page.
- Prefer PNG format.

## README usage

After adding screenshots, link them from the main README using Markdown image syntax.

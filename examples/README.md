# Example Datasets

This directory contains sample datasets for CurveStudio development and testing.

## Files

## KBA105_CMYK_measurement.csv

A sample CMYK measurement dataset for an offset press calibration scenario.

Columns:

- `press`
- `paper`
- `ink_set`
- `channel`
- `nominal_tone`
- `measured_tone`
- `target_printed_tone`
- `notes`

## KBA162_CMYK_measurement.csv

A second sample CMYK measurement dataset for comparison between press conditions.

## sample_tvi_curve.csv

A compact sample table showing nominal tone, measured tone, TVI, target printed tone, and estimated corrected input.

## Notes

These files are example datasets for software development, UI testing, and workflow documentation. They are not intended to replace live press measurement data.

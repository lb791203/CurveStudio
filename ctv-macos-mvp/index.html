1.0.3 统一侧边栏“验证报告”标签

将原“报告与导出”变更为“验证报告”，使侧边栏所有步骤标题均保持一致的 4 字长度，排版更为工整。

1.0.2 优化UI设计系统、统一按钮与提升核心算法精度

重构 CSS 设计变量并整合样式，统一按钮高度与控件尺寸；对齐 G7 灰平衡候选点算法；升级 Yellow 通道保底实地密度；优化补偿曲线渐进式平滑保护；提升测试覆盖率。

1.0.1 优化现场验收报告、导入与测量体验

优化报告布局、标准曲线、统一按钮风格；支持在标准库直接导入 ICC 参考；优化手动测量和视图跳转；重命名报告导出步骤。

0.1.12 G7 / P2P51 / 曲线页打磨

修正 G7 图表、P2P51 导入提示、Lab a*b* 色度图、补偿模拟网点扩大视图和曲线表输出列。

后续计划 稳定版升级通道
